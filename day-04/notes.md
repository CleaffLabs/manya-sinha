# Day 04 - How does n8n handle data?

## Notes
* n8n nodes use data in the form of JSON and Lists.
* JSON stores data as key-value pairs.
* Lists are collections of items/objects.
* JSON objects can be nested (embedded JSON).
* Dot notation is used to access values from JSON.
* One JSON object is similar to one row in a table.
* n8n works with items as node inputs and outputs.
* Nodes generally execute once for each input item.
* Expressions are used to access and manipulate data.
* Expressions can reference item values using `$json`.
* JavaScript can be used inside expressions.
* Expressions can combine dynamic values and static text.
* Input and output data can be viewed in Table, JSON and Schema views.
