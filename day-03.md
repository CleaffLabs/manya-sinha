# Day 03 - n8n Nodes

## Notes
* Nodes are the building blocks of n8n workflows.
* Every workflow is created by connecting nodes.
* Three main categories of nodes:
  * Trigger Nodes
  * Function Nodes
  * Action/App Nodes
* Workflows start with a trigger node.
* Nodes can be searched and added from the canvas.
* Different nodes provide different operations/actions.
* Node settings include:
  * Parameters
  * Credentials
  * Input Data
  * Output Data
* Google Sheets node can be used to read and manage spreadsheet data.
* Node settings can control execution, retries, errors, and notes.
