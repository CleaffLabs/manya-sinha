# Day 07 - Collaboration and Sharing

## Notes
* n8n has an active community for support, feature requests, and workflow discussions.
* Workflow templates can be used instead of building workflows from scratch.
* Three user roles in n8n:

  * Owner
  * Admin
  * Member
* Workflows can be shared with other users for collaboration.
* Workflow roles include Creator and Editor.
* Credential sharing allows users to use credentials without seeing sensitive information.
* API keys and credentials remain secure when shared.
* n8n provides its own API for managing workflows, executions, and credentials.
* The n8n node can be used to interact with n8n programmatically.
