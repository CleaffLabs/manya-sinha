# Day 01 - Introduction to Automation

## Notes
* Automation helps reduce manual work and human error.
* Data-driven decisions are more reliable than intuition-based decisions.
* Automation is a predictable set of predefined actions that transfer data from one point to another.
* Core concepts:
  * Trigger
  * Filter
  * Action
* A workflow generally follows:
  * Trigger → Filter/Transform Data → Action
* Workflows should be mapped out before building them.
* Flowcharts help visualize and plan automations.
